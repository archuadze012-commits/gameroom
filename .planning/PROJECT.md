# Project: Gameroom

Gameroom is a community platform for Georgian gamers, featuring LFG matchmaking, forums, chat, news, and tournaments.

## Vision
To provide a centralized hub for Georgian gamers to find teammates, stay updated on gaming news, participate in tournaments, and build a vibrant community.

## Tech Stack
- **Framework:** Next.js 16 (App Router, Turbopack)
- **Language:** TypeScript 5 + React 19
- **Styling:** Tailwind CSS v4 + shadcn/ui (base-nova preset)
- **Database:** Supabase PostgreSQL
- **Auth:** Supabase Auth (Email magic link, Google, Discord OAuth)
- **ORM:** Drizzle ORM + postgres-js
- **Forms:** react-hook-form + zod
- **Hosting:** Vercel

## Core Features
- **LFG (Looking For Group):** Real-time matchmaking with filters and AI-assisted posting.
- **Social Feed:** User posts, following system, reactions, and mentions.
- **Forums:** Categorized discussions with threaded replies.
- **News:** Gaming articles with comments.
- **Tournaments:** Bracket generation, registration, and result reporting.
- **Messaging:** Real-time chat and direct messages.
- **Gamification:** XP, levels, currency (NC/PRO), and shop.
